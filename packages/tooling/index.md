# Tooling

Developer experience and build-time integrations that are not runtime native modules in the same sense as platform packages.

| Package | Role |
|---------|------|
| [Ambient types (CLI)](/packages/tooling/tamer-ambient-types) | `.tamer/tamer-components.d.ts` and `tsconfig` wiring from **`t4l init`** / **`t4l link`** |
| [tamer-env](/packages/tooling/tamer-env) | Rspeedy plugin: `.env` files and compile-time env injection (`t4l add-core`) |
