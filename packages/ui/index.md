# UI packages

Layout, safe area, system chrome, and icon typography for Lynx screens.

| Package | Role |
|---------|------|
| [tamer-screen](/packages/ui/tamer-screen) | `Screen`, `SafeArea`, `AvoidKeyboard` |
| [tamer-insets](/packages/ui/tamer-insets) | Safe area and keyboard insets |
| [tamer-system-ui](/packages/ui/tamer-system-ui) | Status bar, navigation bar, theme |
| [tamer-icons](/packages/ui/tamer-icons) | Native `<icon>` and font assets |

These are included in **`t4l add-core`** (except optional combinations—see [Packages overview](/packages/)).
